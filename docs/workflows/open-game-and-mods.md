# Workflow: Opening Game and Mods Together

1. Click `Add Game Folder` and choose the desired game root manually.
2. Click `Add DLC Folder` and/or `Add Mods Folder` for any additional roots you want indexed.
3. Run indexing.
4. Use the source-scope checkboxes plus search/facets to browse the combined corpus in either `Asset Browser` or `Raw Resource Browser`.
5. Read the `Source` and `Package` metadata in the details pane to distinguish Game vs DLC vs Mods content.

The current browser keeps Game/DLC/Mods as separate data-source entries while querying them through the same SQLite index.
