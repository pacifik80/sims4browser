# Workflow: Opening Game and Mods Together

1. Click `Update Index`.
2. In the dialog, click `Add Folder` for the game root and any additional DLC or Mods roots you want indexed.
3. Review the stored folder list. The app infers `Game` / `DLC` / `Mods` from each path and keeps the folders for later runs.
4. Optionally adjust `Workers` and the package-read cache memory target, then start indexing.
5. Use the source-scope checkboxes plus search/facets to browse the combined corpus in either `Asset Browser` or `Raw Resource Browser`.
6. Read the `Source` and `Package` metadata in the details pane to distinguish Game vs DLC vs Mods content.

The current browser still keeps Game/DLC/Mods as separate data-source entries for filtering, but the indexing workflow no longer asks the user to pick a source kind manually for each folder.
