# Workflow: Exporting a Build/Buy Object

## Current workflow

1. Add one or more Game or DLC folders.
2. Run indexing.
3. Open the `Logical Assets` tab.
4. Search for the Build/Buy seed resource.
5. Inspect the heuristic asset summary and linked-resource diagnostics.
6. Use raw export on the selected root resource when you need the current implemented export path.

## Current limitation

Full Build/Buy asset-bundle FBX export is not wired to real Sims 4 scene reconstruction yet. The canonical FBX export pipeline exists, but real object reconstruction remains deferred.
