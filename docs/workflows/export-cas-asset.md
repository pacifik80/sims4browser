# Workflow: Exporting a CAS Asset

## Current workflow

1. Add Game/DLC/Mods folders manually.
2. Run indexing.
3. Open the `Logical Assets` tab.
4. Search for the CAS part.
5. Inspect the category guess, root TGI, and linked-resource diagnostics.
6. Use raw export on the selected root resource when you need an export from the current build.

## Current limitation

The current build does not yet reconstruct a real Sims 4 CAS geometry/rig/textured scene for FBX export.
